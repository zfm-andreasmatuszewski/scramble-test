# Scramble Test

## Requirements

- Composer
- NPM
- Access to Scramble Pro

## Initialization

After pulling the repository, you initialize the app via

```bash
cp .env.example .env && \
composer i && \
npm i
```

## Start Application

Start the application using

```bash
php artisan serve
```

Now you can access the Docs created by Scramble via [http://127.0.0.1:8000/docs/api/](http://127.0.0.1:8000/docs/api/).
